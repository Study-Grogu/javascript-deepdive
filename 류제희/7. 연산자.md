# [08] 연산자 (74p-92p)

  하나 이상의 표현식을 대상으로 연산자를 사용하여 하나의 값을 만든다

## 산술 연산자

### 1. 이항 산술 연산자

  2개의 피연산자를 산술 연산
  어떤 산술 연산을 해도 피연산자의 값이 바뀌는 경우는 없고 언제나 새로운 값을 만든다

  ```javascript 
  5+2; // 7
  5-2; // 3
  ```

### 2. 단항 산술 연산자

  증가/감소(++,--) 연산자는 피연산자의 값을 변경하는 부수 효과가 있다
  ㄴ 위치에 의미가 있다.

  ```javascript 
  var x = 5, result;
  // 선할당 후증가
  result = x++;
  console.log(result,x); // 5, 6

  x에 5를 할당한 후에 증가

  // 선증가 후할당
  result = ++x;
  console.log(result,x); // 7, 7
  
  x에 1을 먼저 더한 다음, 그 증가된 값을 바로 반환 후 result에 할당
  ```

  **+ 단항 연산자**

  숫자 타입이 아닌 피연산자에 사용하면, 숫자 타입으로 변환하여 반환
  부수효과는 없다

  ```javascript
  var x = '1';

  // 문자열을 숫자로 타입 변환
  console.log(+x); // 1
  // 부수효과는 없다
  console.log(x); // "1"

  // 불리언 값을 숫자로 타입 변환
  x = true;
  console.log(+x); // 1

  x = false;
  console.log(+x); // 0
  // 부수효과는 없다
  console.log(x); // true

  // 문자열을 숫자로 타입 변환할 수 없으므로 NaN을 반환
  x = 'Hello';
  console.log(+x); // NaN
  // 부수효과는 없다
  console.log(x); // "Hello" 
  ```

  ### 3. 문자열 연결 연산자

  + 단항 연산자는 피연산자 중 하나 이상이 문자열인 경우
  <br>문자열 연결 연산자로 동작

  ```javascript
  // 문자열 연결 연산자
  '1' + 2; // '12'
  
  // true는 1로 타입 변환 
  1 + true; // 2
  // false 0으로 타입 변환 
  1 + false; // 1
  // null 0으로 타입 변환
  1 + null; // 1

  // undefined는 숫자로 타입 변환할 수 없다
  +undefined; // NaN
  1 + undefined // NaN
  ```

  ㄴ 암묵적 타입 변환 => 자세한 내용은 9장에서 참고


  ## 할당 연산자

  좌항의 변수에 값을 할당
  <br>좌항의 변수에 값을 할당하므로 변수 값이 변하는 부수 효과가 있다

  =, +=, -=, *= ... 

  ## 비교 연산자

  좌항과 우항의 피연산자 비교 후 결과를 불리언 값으로 반환

  ### 1. 동등 비교 연산자 

  ==
  <br>x와 y의 값이 같음

  ```javascript
  // 동등 비교
  5 == 5; // true

  // 타입은 다르지만 암묵적 타입 변환을 통해 타입을 일치시키면 동등
  5 == '5' // true
  ```
  
  동등 비교(==) 연산자는 좌항과 우항을 비교할 때 암묵적 타입 변환을 통해 타입을 일치시킨 후 같은 값인지 비교한다

  ### 2. 일치 비교 연산자

  === 
  <br>x와 y의 값과 타입이 같음
  ```javascript
  // 일치 비교
  5 === 5; // true

  // 암묵적 타입 변환을 하지 않고 값을 비교
  5 === '5'; // false
  ```

  일치 비교(===) 연산자는 좌항과 우항의 타입도 같고 값도 같은 경우에 한하여 true를 반환한다

  **일치 비교 연산자에서 주의할 것**

  NaN

  ```javascript
  // NaN은 자신과 일치하지 않는 유일한 값
  NaN === NaN; // false
  ```

  숫자 0

  ```javascript
  // 양의 0과 음의 0의 비교. 일치 비교/동등 비교 모두 결과는 true
  0 === -0; // true
  0 == -0; // true
  ```

  ### 3. 대소 관계 비교 연산자
  
  피연산자의 크기를 비교하여 불리언 값을 반환


  ## 삼항 조건 연산자

  조건식의 평가 결과에 따라 반환할 값을 결정
  <br>조건식 ? 조건식이 true일 때 반환할 값 : 조건식이 false일 때 반환할 값
  <br>피연산자1 ? 피연산자2 : 피연산자3

  **조건식의 평가 결과가 불리언 값이 아닐 경우 불리언 값으로 암묵적 타입 변환**

  ```javascript
  var x = 2;

  // 2 % 2는 0이고 0은 false로 암묵적 타입 변환
  var result = x % 2 ? '홀수' : '짝수';
  console.log(result); // 짝수
  ```

  삼항 조건 연산자 표현식은 값으로 평가할 수 있는 표현식인 문이다.

  ## 논리 연산자

  우항과 좌항의 피연사자를 논리 연산
  
  ```javascript 
  // 논리합(||) 연산자 - 하나만 일치해도 true
  true || false; // true
  false || false; // false

  // 논리곱(&&) 연산자 - 둘 다 일치해야 true
  true && true; // true
  true && false; // false
  false && false; // false

  // 논리 부정(!) 연산자 - 언제나 불리언 값을 반환 
  // 만약 피연산자가 불리언 값이 아니면 불리언 타입으로 암묵적 타입 변환
  !true; // false
  !false; // true 


  // 논리합(||) 또는 논리 곱(&&) 연산자 표현식의 평가 결과는 불리언 값이 아닐수도 있다
  // 언제나 2개의 피연산자 중 어느 한쪽으로 평가
  'Cat' && 'Dog'; // 'Dog'
  ```


  ## 그룹 연산자

  소괄호 ('()')로 피연산자를 감싸는 그룹 연산자를 사용하여 연산자의 우선순위 조절
  <br>연산자 우선순위가 가장 높다

  ## typeof 연산자

  피연산자의 데이터 타입을 문자열로 반환
  **null은 object를 반환한다**

  선언하지 않은 식별자를 typeof 연산자로 연산하면 undefined를 반환
  ```javascript
  // undeclared 식별자를 선언한 적이 없다
  typeof undeclared; // undefined
  ```

## 연산자의 부수 효과

부수효과가 있는 연산자는 다른 코드에 영향을 주는 할당 연산자(=), 증가/감소 연산자(++/--), delete 연산자다

```javascript
var x;

// 할당 연산자는 변수 값이 변하는 부수 효과가 있다
x = 1;
console.log(x); // 1

// 증가/감소 연산자(++,--)는 피연산자의 값을 변경하는 부수 효과가 있다
x++;
console.log(x); // 2

// delete 연산자는 객체의 프로퍼티를 삭제하는 부수 효과가 있다
var o = { a: 1 };
delete o.a;
console.log(o); // {}
```

이는 모두 해당 변수나 객체를 사용하는 다른 코드에 영향을 준다

  ### 1. 삼항 연산자와 if문의 차이

  삼항 연산자 표현식은 값처럼 사용할 수 있지만
  <br>if..else문은 값처럼 사용할 수 없다

  삼항 연산자는 조건에 따라 값을 반환하는 간단한 조건 표현식에 적합하고 변수에 바로 할당할 때 유용하다
  <br>if문은 여러 문장을 포함하거나 복잡한 로직을 처리할 때 적합하며, 명확한 블록 구조를 제공한다


  ```javascript
  // 삼항 연산자는 표현식이기 때문에 값을 반환해서 변수에 바로 대입하거나 함수 인자로 쓸 수 있다
  var x = 10;
  var result = x % 2 ? '홀수' : '짝수';
  console.log(x); // 짝수

  // `x % 2 ? '홀수' : '짝수'` 이 표현식(값)을 변수 'x'에 바로 할당함
  // 함수 인자도 가능
  function printOddOrEven(num) {
    console.log(num);
  }
  printOddOrEven(x % 2 ? '홀수' : '짝수');
  // 짝수
  ```
  

  if문은 문(statement)이어서 값을 직접 반환하거나 변수 대입에 쓸 수 없다
  ```javascript
  var x = 10;

  var result = if (x % 2) {
  result = '홀수';
  } else {
  result = '짝수';
  };
  ```

  삼항 연산자 : 표현식(값으로 평가)
  <br>if문 : 표현식이 아닌 문(값으로 평가X)

  ***조건에 따라 어떤 '값'을 결정해야 한다면(어떤 값/결과를 구하거나 선택해야 할 때) 삼항 연산자, 
  <br>조건에 따라 수행해야 할 문이 여러개라면(여러개의 실행 문장을 수행) if..els문이 더 가독성이 좋다***

  ```javascript
  //수행해야 할 문이 여러개 있는 상황/ 여러개의 실행 문장을 수행해야 할 때

if (score >= 90) {
  console.log("A학점");
  ```

  ### 2. Number.isNaN 

  Number.isNaN()은 지정한 값이 정확히 NaN인 경우에만 'true'를 반환
  ```javascript
  Number.isNaN(NaN); // true
  Number.isNaN(10); // false
  
  Number.isNaN(1+undefined); // true
  // '1+undefined' 연산을 하면 숫자형으로 강제 변환
  // 'undefined'를 숫자로 변환하면 'NaN'이 됨
  ```
