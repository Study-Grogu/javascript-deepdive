# 모던 자바스크립트 Deep Dive

모던 자바스크립트 Deep Dive 스터디 기록입니다.

모던 자바스크립트 Deep Dive(위키북스)와 프런트엔드 레벨을 높이는 자바스크립트 퀴즈북(한빛미디어)를 활용하여 자바스크립트에 대한 이해를 높입니다.

각자의 디렉토리에 한 주간 한 회차의 분량 중 마크다운 문서와 예제파일로 배운 내용과 생각할 점을 정리합니다.

1. 한 주 동안 공부한 것을 정리하여 레포에 커밋
2. 각자 중요하다고 생각하는 부분을 정리하여 커밋
3. 모르는 부분도 질문을 정리하여 커밋

4. 돌아가며 주 담당 선정,  자유롭게 주제 등에 대해 해당 주차 주도
   - 순번 (이유리 -> 류제희 -> 임경민 -> 김윤기 -> 박창기)
   - 주 진행 예시
     1) 주 담당이 레포에 올라온 모든 것을 확인
     2) 여기서 중요하다고 생각하는 것에 대해 이야기 하기
     3) 질문이 있는 사람은 질문
6. 매주 목요일 20시 30분 부터 최대 3 시간
