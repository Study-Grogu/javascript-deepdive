# 모던 자바스크립트 딥다이브

모던 자바스크립트 딥다이브 스터디 기록입니다.

각자의 디렉토리에 마크다운 문서와 예제파일로 배운 내용과 생각할 점을 정리합니다.

## 진행기록

- 1회차(25년 10월 1일): 책을 전체적으로 살펴보고 앞으로의 스터디 계획을 세웠습니다. 1장 프로그래밍부터 6장 데이터 타입까지 살펴보고 이야기를 나눴습니다.
- 2회차(25년 10월 15일): 7장 연산자부터 11장 원시 값과 객체의 비교에 대해 공부한 것을 공유하고 이야기를 나눴습니다.
  - 류제희: 7장 연산자, 8장 제어문
  - 임경민: 9장 타입 변환과 단축 평가
  - 백수현: 10장 객체 리터럴, 11장 원시 값과 객체의 비교
- 3회차(25년 10월 21일 예정)
  - 류제희: 12장 함수 .1 함수란 ~ .5 함수호출
  - 백수현: 12장 함수 .6 참조에 의한 전달과 외부 상태의 변경 ~ .7 다양한 함수의 형태
  - 임경민: 13장 스코프
