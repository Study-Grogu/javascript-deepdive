# 모던 자바스크립트 딥다이브

모던 자바스크립트 딥다이브 스터디 기록입니다.

각자의 디렉토리에 마크다운 문서와 예제파일로 배운 내용과 생각할 점을 정리합니다.

## 진행기록

- 1회차(25년 10월 1일): 책을 전체적으로 살펴보고 앞으로의 스터디 계획을 세웠습니다. 1장 프로그래밍부터 6장 데이터 타입까지 살펴보고 이야기를 나눴습니다.
- 2회차(25년 10월 14일 예정): 7장 연산자부터 11장 원시 값과 객체의 비교까지 다룹니다.
  - 류제희: 7장 연산자, 8장 제어문
  - 임경민: 9장 타입 변환과 단축 평가
  - 백수현: 10장 객체 리터럴, 11장 원시 값과 객체의 비교
